# coaching/games/

This folder contains individual game film review pages for Oliver Crane.

## Naming convention
`YYYY-MM-DD-opponent-name.html`

Example: `2025-05-18-west-side.html`

## How to add a new game review
1. Copy `coaching/template.html` to this folder with the correct filename
2. Fill in the stats, video embed, coaching notes, and film breakdown
3. Add an entry to the `sessions` array in `index.html` with `file: "coaching/games/YYYY-MM-DD-opponent.html"`
